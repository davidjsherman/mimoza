Welcome to Mimoza

Mimoza is a navigation system that allows to explore metabolic models in a semantically zoomable manner.

  • Description
  • Try it!
  • Download
  • Examples

Mimoza creates a 3-level representation of your model:

  • The general view represents the compartments of the model and the transport reactions between them;
  • The intermediate view shows the generalized versions of reactions and metabolites in each compartment;
  • The most detailed view represents the initial model with the generalization-based layout (where similar metabolites and reactions are placed next to each other).

Try Mimoza now by submitting your model (in SBML format) in the form below.

(Available for models of a modest size (hundreds of reactions). For large ones either download Mimoza to install it locally, or upload a generalized model.)

Mimoza.zip - download the Mimoza code. You can install it locally, or as a Galaxy tool (download the config file).

The README file inside the Mimoza.zip archive provides the installation and usage instructions.

Mimoza is powered by Leaflet, Model generalization library, ChEBI, Tulip, GeoJSON, jQuery, Python, libSBML, and PyCharm.
