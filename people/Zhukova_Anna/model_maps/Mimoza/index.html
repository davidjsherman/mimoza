Welcome to Mimoza

Mimoza is a navigation system that allows to explore metabolic models in a semantically zoomable manner:

  • The general views represent the compartments of the model and the transport reactions between them;
  • the intermediate view shows the generalized versions of reactions and metabolites in each compartment;
  • the most detailed view represents the initial model with the generalization-based layout (where similar metabolites and reactions are placed next to each other).

Try Mimoza now by submitting your model (in SBML format) in the form below. (Available for the models of a modest size (hundreds of reactions). For large ones either install Mimoza locally, or upload a generalized model.)

You can also download the Mimoza's code to install it locally, or as a Galaxy tool (download the config file).
